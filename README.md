# Smart-Ambulance-Traffic-Control-System
Smart Ambulance Traffic Signal Override System is an IoT-based intelligent traffic management solution designed to reduce ambulance response time during emergencies. The system enables an approaching ambulance to communicate with nearby traffic signals using wireless communication and automatically provides a green signal along its route. 
description (how it works) • The ambulance unit reads a siren sensor and GPS, and when siren is active it sends an override message (and ambulance metadata) using a long-range radio (LoRa) or serial radio to the nearby traffic signal controller and the hospital display unit. • The traffic unit receives messages from ambulance(s), overrides the normal traffic cycle to show green in the ambulance direction, displays ambulance metadata (on OLED / LEDs) and then returns to the normal cycle after the ambulance passes. • The hospital unit receives the ambulance message (location & patient info) and displays it for staff readiness; it can also generate a buzzer alert. • All override events and telemetry are logged to a cloud DB for records/analysis.
